Clued Up, and why you should choose our game

We obviously want you to select our game, but we have many genuine reasons why it will be simple for you to get along with our software even if you haven't used Unity yourself.

TLDR;

  • We tell you what to edit if you need pointers
  • Our code is layed out sensibly, and heavily documented
  • Don't let 3D models scare you off, Adobe Fuse (free) gives you access to them within 30 seconds of drag and drop action

WE'VE ALREADY DONE THE WORK FOR YOU

For assessment 3 our code will need to be extended, and doing this with a shoddy project is a bad idea.

As a helping hand, we have outlines simple methods of extending our code if you choose our project so that less time is spent tryin to work out what happens where. If that is not enough, we will happily help you understand what each of the scripts does, and what happend in the Unity Editor vs the scripts if you are unfamilliar with Unity.

How to do assessment 3:

  1. Scoring:

    Scoring can easily be implemented by storing Time.time when RoomController.Start() is called (only when with roomIndex = 0) into a new property of Story, and having another new property of Story contain the total of all time faults. The current score could then be calculated at any time based on a function that uses the current Time.time and the start time stored in Story to work out total time playing, and the deductable points.

WE WILL GIVE YOU EVERYTHING

We have saved all the original Photoshop background files, character models and diagrams that we will give you if you choose our project.

This means if there are any changes you ever need to make to assets in our game, you will have access to the original production projects. [Provided you have the correct software - namely Photoshop, Adobe Fuse (free), and StarUML (free)].

"Definately the best graphics of any of the SEPR games - anonymous SEPR demonstrator

One big advantage of using unity is how easy graphics are... Lighting is done through a GUI, animations are drag and drop, and movement of our detectives is done through graphical state machines.

Assessment 3 requires that you add more characters to the mix, which (if you want to continue the appearence of Clued Up) will require you to make 3D models.

THEY TAKE LITERALLY LIKE 5 MINUTES

^ not joking. Adobe Fuse (free) is a drag and drop 3D model maker that exports to Mixamo (free) for rigging and downloading for unity. This process is super super simple and it guides you along, and simple tutorial exist if you cannot understand what to do (I didn't need them tbh).

Importing them to unity is simple too, but if you choose our project I will help you add an idle animation to them so they dont just look like a starfish. This can easily be done, and you could probably also work it out if you don't want my help.
